# Resume Folder

Place your resume PDF file in this folder.

## How to Add Your Resume:

1. **Create/Export your resume as PDF**
   - Name it: `resume.pdf`

2. **Add the file to this folder**
   - Copy your `resume.pdf` to: `portfolio/assets/resume.pdf`

3. **The resume download links will automatically work**
   - Navigation bar "Resume" button
   - Hero section "Download Resume" button
   - Footer "Download Resume" button

## File Structure:
```
portfolio/
├── assets/
│   ├── css/
│   ├── js/
│   └── resume.pdf    ← Your resume goes here
└── index.html
```

## Tips:
- Keep the file size under 2MB for faster loading
- Use a professional filename like `resume.pdf` or `Priya_Bhanu_Resume.pdf`
- Make sure it's in PDF format for universal compatibility
- Test the download link after deploying to GitHub Pages
