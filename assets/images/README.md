### How to add your profile picture:

1. **Get a professional photo of yourself**
   - Use a clear, professional headshot
   - Recommended size: 400x400 pixels (square)
   - Format: JPG or PNG

2. **Name and save the file**
   - Name it: `profile.jpg` (or `profile.png`)
   - Save it in this folder: `/home/bhanup/Main/Projects/portfolio/assets/images/`

3. **Copy your photo**
   ```bash
   cp /path/to/your/photo.jpg /home/bhanup/Main/Projects/portfolio/assets/images/profile.jpg
   ```

4. **Verify the file**
   ```bash
   ls -lh /home/bhanup/Main/Projects/portfolio/assets/images/
   ```

### Tips for a good profile picture:
- Use good lighting (natural light is best)
- Face the camera directly or at a slight angle
- Wear professional attire
- Use a neutral or blurred background
- Smile naturally
- High quality but optimized (under 200KB)
